LoGinKeyboard v3.1.2

배포일: 2026-07-02

변경 사항:
- CapsLock 이동 레이어의 방향키 전송 방식을 수정했습니다.
- `Shift + CapsLock + J/I/L/K`가 실제 `Shift + 방향키`처럼 선택 영역을 확장하도록 했습니다.
- `Ctrl + CapsLock + J/L`과 `Ctrl + Shift + CapsLock + J/L`도 실제 방향키 조합처럼 동작하도록 했습니다.
- 내부적으로 조합 문자열을 새로 보내지 않고, 방향키만 눌렀다 떼서 현재 물리 보조키 상태를 Windows가 그대로 적용하게 했습니다.

배포 파일:
- LoGinKeyboard-v3.1.2.exe

주의 사항:
- 키보드 훅이 동작하지 않으면 관리자 권한으로 실행하세요.
- 키보드 훅을 사용하는 앱이라 Windows Defender나 백신이 경고할 수 있습니다.
