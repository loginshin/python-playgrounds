LoGinKeyboard v3.1.7

배포일: 2026-07-12

변경 사항:
- Right Ctrl을 일반 보조키가 아닌 완전히 억제된 전용 레이어 키로 변경했습니다.
- Right Ctrl down이 Windows에 전달되지 않으므로 방향키 사용 후 Ctrl 상태가 남지 않습니다.
- Right Ctrl 방향키, 검색, 번역, 도움말 기능은 별도 물리 키 상태 추적으로 유지됩니다.
- Right Ctrl을 누를 때 이전 프로세스에서 남았을 수 있는 Ctrl 상태도 즉시 해제합니다.

동작 변경:
- Right Ctrl 단독 입력은 더 이상 일반 Ctrl 키로 동작하지 않습니다.

배포 파일:
- LoGinKeyboard-v3.1.7.exe

주의 사항:
- 기존 버전을 시스템 트레이에서 완전히 종료한 뒤 새 버전을 실행하세요.
- 이 EXE는 Python 3.14 64-bit로 빌드한 Windows 10/11용 배포 파일입니다.
