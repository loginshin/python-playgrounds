LoGinKeyboard v3.1.3

배포일: 2026-07-02

변경 사항:
- CapsLock 이동 레이어의 방향키 전송을 Windows API 기반 확장 키 입력으로 변경했습니다.
- `Shift + CapsLock + J/I/L/K`가 실제 `Shift + 방향키`처럼 텍스트 선택을 확장하도록 다시 수정했습니다.
- `Ctrl + Shift + CapsLock + J/L`도 실제 `Ctrl + Shift + Left/Right`처럼 단어 단위 선택 확장이 되도록 개선했습니다.
- `CapsLock + H + J/L/I/K`의 `Home`, `End`, `PageUp`, `PageDown`도 확장 키로 전송합니다.

배포 파일:
- LoGinKeyboard-v3.1.3.exe

주의 사항:
- 키보드 훅이 동작하지 않으면 관리자 권한으로 실행하세요.
- 키보드 훅을 사용하는 앱이라 Windows Defender나 백신이 경고할 수 있습니다.
