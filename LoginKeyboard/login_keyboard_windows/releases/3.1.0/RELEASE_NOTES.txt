LoGinKeyboard v3.1.0

배포일: 2026-07-02

변경 사항:
- Windows 시스템 트레이 아이콘을 추가했습니다.
- 트레이 우클릭 메뉴에 도움말 열기, 블로그 열기, 종료 기능을 추가했습니다.
- `Ctrl + Shift + Q` 빠른 종료 단축키를 유지했습니다.
- `pystray`/`Pillow` 의존성을 제거하고 Windows 네이티브 트레이 API로 바꿔 exe 용량을 줄였습니다.

배포 파일:
- LoGinKeyboard-v3.1.0.exe

주의 사항:
- 키보드 훅이 동작하지 않으면 관리자 권한으로 실행하세요.
- 키보드 훅을 사용하는 앱이라 Windows Defender나 백신이 경고할 수 있습니다.
