LoGinKeyboard v3.1.1

배포일: 2026-07-02

변경 사항:
- CapsLock 이동 레이어에서 `Shift`, `Ctrl`, `Alt` 보조키를 유지하도록 수정했습니다.
- 이제 `Shift + CapsLock + J/I/L/K`는 실제 `Shift + 방향키`처럼 선택 영역을 확장합니다.
- 이제 `Ctrl + CapsLock + J/L`은 실제 `Ctrl + Left/Right`처럼 단어 단위 이동합니다.
- 이제 `Ctrl + Shift + CapsLock + J/L`은 실제 `Ctrl + Shift + Left/Right`처럼 단어 단위 선택을 확장합니다.
- `CapsLock + H + J/L/I/K`의 `Home`, `End`, `PageUp`, `PageDown` 이동에도 같은 보조키 보존을 적용했습니다.

배포 파일:
- LoGinKeyboard-v3.1.1.exe

주의 사항:
- 키보드 훅이 동작하지 않으면 관리자 권한으로 실행하세요.
- 키보드 훅을 사용하는 앱이라 Windows Defender나 백신이 경고할 수 있습니다.
