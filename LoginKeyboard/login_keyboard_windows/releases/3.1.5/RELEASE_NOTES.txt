LoGinKeyboard v3.1.5

배포일: 2026-07-12

변경 사항:
- CapsLock + Space/Z/X/C/A/S/D/Q/W/E가 일반 숫자 문자열 대신 실제 넘버패드 0-9의 물리 스캔 코드를 입력하도록 변경했습니다.
- Right Ctrl + 방향키가 실제 Home/End/PageUp/PageDown 키를 입력하도록 변경했습니다.
- Right Ctrl 방향 레이어 사용 후 Ctrl 및 방향키 상태가 남아 후속 입력이 불안정해질 수 있던 문제를 수정했습니다.
- 실행 방법과 단일 EXE 빌드 방법을 README에 정리했습니다.

배포 파일:
- LoGinKeyboard-v3.1.5.exe

주의 사항:
- 이 EXE는 Python 3.14 64-bit로 빌드한 Windows 10/11용 배포 파일입니다.
- 키보드 훅이 동작하지 않으면 관리자 권한으로 실행하세요.
- 키보드 훅을 사용하는 앱이라 Windows Defender나 백신이 경고할 수 있습니다.
