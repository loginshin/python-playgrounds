Input Language Indicator v0.3.0
================================

- 작업표시줄 오른쪽 숨겨진 아이콘(^) 영역에 상주 아이콘 추가
- 트레이 아이콘에서 현재 한/영 입력 상태 표시
- 트레이 아이콘 우클릭 메뉴에서 안전하게 종료 가능
- 최신 브라우저와 앱을 위한 접근성 API 커서 위치 탐색 추가
- 일반 Win32 입력창과 최신 입력창 양쪽에서 커서 추적

실행:
InputLanguageIndicator.exe를 더블 클릭합니다.

종료:
작업표시줄 오른쪽의 아이콘을 우클릭하고 '종료'를 누릅니다.
